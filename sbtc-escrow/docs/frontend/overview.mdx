# Frontend Overview

The sBTC Escrow frontend is a React single-page application that provides a complete UI for managing escrows, viewing analytics, and administering the contract.

## Tech Stack

| Technology | Version | Purpose |
|-----------|---------|---------|
| React | 18 | UI framework |
| React Router | v6 | Client-side routing |
| TypeScript | 5.8 | Type safety |
| Vite | 5.4 | Build & dev server |
| Stacks Connect | Latest | Wallet integration |
| TanStack React Query | Latest | Data fetching & caching |
| Supabase JS | Latest | Realtime subscriptions |
| Radix UI | 24+ | Accessible component primitives |
| Tailwind CSS | 3.4 | Utility-first styling |
| Framer Motion | Latest | Animations |
| Sonner | Latest | Toast notifications |

## Pages

| Route | Page | Description |
|-------|------|-------------|
| `/` | Landing | Marketing and onboarding |
| `/dashboard` | Dashboard | User overview with active escrows |
| `/create` | CreateEscrow | Form to create new escrows |
| `/escrows` | MyEscrows | List all user escrows |
| `/escrow/:id` | EscrowDetail | Single escrow with action buttons |
| `/activity` | Activity | Global event feed |
| `/analytics` | Analytics | Platform statistics |
| `/settings` | Settings | User preferences |
| `/admin` | AdminDashboard | Admin overview |
| `/admin/disputes` | DisputeQueue | Manage disputes |
| `/admin/controls` | ContractControls | Fees, pause, ownership |
