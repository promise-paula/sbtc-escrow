# SDK Overview

The `sbtc-escrow-sdk` is a TypeScript client for interacting with the sBTC Escrow smart contract from Node.js or browser environments.

## Features

- **Read-only queries** вҖ” Fetch escrows, stats, and config without a wallet
- **Transaction building** вҖ” Create, release, refund, dispute escrows with signing
- **Admin operations** вҖ” Manage fees, disputes, and ownership
- **Network-aware** вҖ” Auto-configures for testnet or mainnet
- **Post-conditions** вҖ” Built-in transaction safety for STX and sBTC
- **TypeScript-first** вҖ” Full type definitions for all methods and return values

## Quick Example

```typescript
import { EscrowClient, TokenType } from 'sbtc-escrow-sdk';

// Initialize client
const client = new EscrowClient({ network: 'testnet' });

// Read (no signing)
const config = await client.getConfig();
const escrow = await client.getEscrow(1);
const stats = await client.getPlatformStats();

// Write (requires private key)
const result = await client.createEscrow(
  {
    seller: 'ST2CY5V39NHDPWSXMW9QDT3HC3GD6Q6XX4CFRK9AG',
    amount: 1_000_000,
    description: 'Payment for services',
    durationBlocks: 960,
    tokenType: TokenType.STX,
  },
  { senderKey: 'your-private-key-hex' }
);

console.log('TX:', result.txid);
```
