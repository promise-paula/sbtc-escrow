# Smart Contract Overview

The `escrow-v5` contract is a Clarity v4 smart contract deployed on the Stacks blockchain. It manages the full lifecycle of trustless escrow transactions in both STX and sBTC.

## Contract Identity

| Property | Value |
|----------|-------|
| **Name** | `escrow-v5` |
| **Clarity Version** | 4 |
| **Epoch** | 3.3 (post-Nakamoto) |
| **Testnet** | `ST1HK6H018TMMZ1BZPS1QMJZE9WPA7B93T8ZHV94N.escrow-v5` |
| **Mainnet** | TBD |

## Dependencies

| Contract | Purpose |
|----------|---------|
| `sbtc-token` (SIP-010) | sBTC fungible token transfers |

## Version History

| Version | Contract | Features |
|:-------:|----------|----------|
| **V3** | `escrow.clar` | STX-only escrow, dispute resolution, timeout recovery |
| **V4/V5** | `escrow-v5.clar` | Dual-token (STX + sBTC), per-token stats, SIP-010 integration |
